# TicTacToe [![Travis CI Build status](https://travis-ci.org/balaji/TicTacToe.svg?branch=master "Travis CI")](https://travis-ci.org/balaji/TicTacToe)

Browser based game of Tic-Tac-Toe.

### Prerequisites
* ruby `v2.3.0`
* Bundler
* npm - to edit JavaScript files.
* Browserify

### Local machine setup

```
$ bundle install
$ bundle exec rackup
```
